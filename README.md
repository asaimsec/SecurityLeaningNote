# TryHackMe 学習記録

このリポジトリは **TryHackMe（THM）** で学習した内容を整理・記録するためのものです。  
CTFやハンズオンを通して得た知識をアウトプットし、理解を深めることを目的としています。

---

## 🎯 目的
- TryHackMeで学んだ内容の整理・復習
- セキュリティ分野の知識を体系的にまとめる
- 技術的な内容を文章で説明する練習
- 将来的に見返せる自分用ナレッジベースの作成

## 📘 内容について

### 🔹 Writeups（部屋ごとの記録）
- TryHackMeの各Roomの学習メモ
- 攻撃・調査の**手順や考え方**を重視
- フラグや直接的な答えは記載しません

### 🔹 Techniques（技術メモ）
- SQL Injection / XSS / Privilege Escalation などの攻撃手法
- 複数のRoomで共通する知識をまとめたもの
- CTF以外でも使える形で整理

### 🔹 Tools（ツールメモ）
- Nmap / Burp Suite / Metasploit などの使い方
- よく使うコマンドやオプション
- TryHackMeで実際に使った場面のメモ

### 🔹 Cheatsheets（チートシート）
- よく使うLinuxコマンド
- Web攻撃の基本ペイロード
- リバースシェルなどの即確認用メモ

---

## ⚠️ 注意事項
- このリポジトリには以下を含みません
  - フラグ
  - Roomの答えそのもの
  - TryHackMeの規約に反する情報
- 記載内容は**学習目的のみ**で作成しています
- 攻撃手法の理解を目的としており、悪用を意図するものではありません
